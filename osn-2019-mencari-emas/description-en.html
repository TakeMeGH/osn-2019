Description

The city of Manado can be represented by squares of R rows (numbered 1 to R) and C columns (numbered 1 to C). The square in the i-th row and the j-th column is denoted as the square (i, j).

There are two pieces of gold in Manado, located in the square (Ai, Aj) and in the square (Bi, Bj). Both pieces of gold might be located in the same square.

Pak Dengklek wants to find the location of the pieces of gold. Therefore, Pak Dengklek buys a gold detector machine. To use it, Pak Dengklek can stand in one of the squares and push the button on the gold detector machine. If Pak Dengklek is in the square (Di, Dj) and press the button on the gold detector machine, the machine will tell the Manhattan distance to the nearer piece of gold, which is the lesser value between |Di - Ai| + |Dj - Aj| or |Di - Bi| + |Dj - Bj|.

Help Pak Dengklek to find the location of the pieces of gold by pressing the button on the gold detector machine not more than Q times.

Problem Type Information

This is an "interactive" problem. In this problem, you will interact with the judge's program through standard input (stdin) and standard output (stdout). See the following interaction format carefully.

Interaction Format

In the beginning, the values of R, C, and Q will be given from standard input in the following format: 

R C Q

Then, not more than Q times, you can output to standard output in the following format: 

? Di Dj

This means Pak Dengklek will stand in the square (Di, Dj) and press the button on the gold detector machine. The values of Di and Dj have to satisfy 1 ≤ Di ≤ R and 1 ≤ Dj ≤ C. The value which will be told by the gold detector machine (denoted by the value K) will be given from standard input in the following format: 

K

In the end, your answer has to be outputted to standard output in the following format: 

! A'i A'j B'i B'j

This means that you guess that the pieces of gold are located in the square (A'i, A'j) and square (B'i, B'j). Your answer will be considered as correct if at least one of the following condition is satisfied:

  • (Ai, Aj) = (A'i, A'j) and (Bi, Bj) = (B'i, B'j), or
  • (Ai, Aj) = (B'i, B'j) and (Bi, Bj) = (A'i, A'j).

Note

For each test case, denote P as the number of button presses on the gold detector machine. If you correctly guess the location of the pieces of gold and P ≤ Q, then you will receive an AC verdict. If P > Q, then you will receive a WA verdict and a "P > Q" message on the grader. If you do not correctly guess the location of the pieces of gold, or the values of Di and Dj that you output does not satisfy 1 ≤ Di ≤ R and 1 ≤ Dj ≤ C, then you will receive a WA verdict and no message on the grader.

Warning

Always flush after each output.

  • Pascal:
    flush(output);
  • C/C++:
    • for stdio.h (cstdio) library:
      fflush(stdout);
    • for iostream library:
      std::cout << std::flush;
