Description

The OSN competition will begin soon! Chairs have been arranged to form a rectangle containing R rows (numbered 1 to R from north to south) × C columns (numbered 1 to C from west to east) inside the contest hall. The chair in the i-th row and the j-th column is provided for an OSN contestant if (i + j) is odd and provided for an OSN invigilator if (i + j) is even.

Each chair provided for an OSN invigilator is still empty at this point. Each chair provided for an OSN contestant might be empty or be occupied by an OSN contestant. There is at least one OSN contestant.

The condition of all chairs in the contest hall at this point can be represented by a matrix S. If the chair in the i-th row and the j-th column is empty at this point, then S[i][j] contains a character .. Otherwise, if the chair in the i-th row and the j-th column is occupied by a contestant at this point, then S[i][j] contains a character x.

You want to invigilate all the OSN contestants. Therefore, for all chairs provided for an OSN invigilator, you can put an invigilator and determine the sitting direction of the invigilator. Invigilator sitting in the i-th row and the j-th column can invigilate:

  • An OSN contestant sitting in the (i + 1)-th row and the j-th column, if the invigilator sits to the south.
  • An OSN contestant sitting in the (i - 1)-th row and the j-th column, if the invigilator sits to the north.
  • An OSN contestant sitting in the i-th row and the (j + 1)-th column, if the invigilator sits to the east.
  • An OSN contestant sitting in the i-th row and the (j - 1)-th column, if the invigilator sits to the west.

Your score will depend on the number of OSN contestants that are invigilated by at least one invigilator, using a formula that will be explained in the Scoring section.

Problem Type Information

This is an "output-only" problem. For each test case, you write the output to a file.

The input files for this problem can be downloaded here. Inside the .zip file, there are 1 + 5 input files to solve: mengawasi_sample_1.in, mengawasi_1.in, mengawasi_2.in, ..., mengawasi_5.in. The sample input (mengawasi_sample_1.in) is not included in the final score.

For each input file that you solve (you don't have to solve all input files), create an output file mengawasi_sample_X.out (for sample input) or mengawasi_X.out, where X is the test case number. Then, compress all files into a single .zip file, and submit.

Input Format

The input is given with the following format: 

R C
S[1][1]S[1][2]...S[1][C]
S[2][1]S[2][2]...S[2][C]
.             .  .
.              . .
.               ..
S[R][1]S[R][2]...S[R][C]

Output Format

R lines: the i-th line contains C characters. The j-th character in the i-th line is:

  • . if the chair in the i-th row and the j-th column is initially empty and you let it to be empty.
  • x if the chair in the i-th row and the j-th column is occupied by an OSN contestant.
  • v if the chair in the i-th row and the j-th column is initially empty and you put an invigilator on the chair sitting to the south.
  • ^ if the chair in the i-th row and the j-th column is initially empty and you put an invigilator on the chair sitting to the north.
  • > if the chair in the i-th row and the j-th column is initially empty and you put an invigilator on the chair sitting to the east.
  • < if the chair in the i-th row and the j-th column is initially empty and you put an invigilator on the chair sitting to the west.

Scoring

Your output will achieve a score on a test case if your output follows the specified output format and satisfies all the following conditions:

  • If S[i][j] = . and (i + j) is odd, then the j-th character in the i-th row of your output is ..
  • If S[i][j] = . and (i + j) is even, then the j-th character in the i-th row of your output is either ., v, ^, >, or <.
  • If S[i][j] = x, then the j-th character in the i-th row of your output is x.

If P is the number of OSN contestants that are invigilated by at least one invigilator in your solution, and Q is the number of maximum OSN contestants that can be invigilated by at least one invigilator in an optimal solution, then your score is:

Condition Points
P = Q20
P < Q

Test Cases

For all test cases:

  • 2 ≤ R, C ≤ 1,000.
  • S contains only the characters . or x.
  • There is at least one character x in S.
  • If (i + j) is even, then S[i][j] = ..
