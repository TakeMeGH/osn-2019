Description

The OSN competition will begin soon! Chairs have been arranged to form a rectangle containing R rows (numbered 1 to R) × C columns (numbered 1 to C) inside the contest hall. Each chair might be empty or be occupied by an OSN contestant.

The condition of all chairs in the contest hall at this point can be represented by a matrix S. If the chair in the i-th row and the j-th column is empty at this point, then S[i][j] contains a character 0. Otherwise, if the chair in the i-th row and the j-th column is occupied by a contestant at this point, then S[i][j] contains a character 1.

Pak Dengklek wants to brief the OSN contestants. Looking at the distributed seating of the OSN contestants, Pak Dengklek will gather them first before the briefing. Pak Dengklek wants all contestants to be gathered in one rectangle. In other words, all contestants sit between the r1-th to r2-th row and the c1-th to c2-th column (for some value 1 ≤ r1 ≤ r2 ≤ R and 1 ≤ c1 ≤ c2 ≤ C), and there is no empty chair inside the mentioned row and column ranges.

To gather the contestants, Pak Dengklek might have to move some of the contestants to any empty seat. Pak Dengklek wants to move as few contestants as possible since each movement disturbs other contestants. Pak Dengklek is wondering about the minimum number of movements that have to be done for all contestants to be gathered in one rectangle, or if it is impossible to do so.

Input Format

The input is given with the following format: 

R C
S[1][1]S[1][2]...S[1][C]
S[2][1]S[2][2]...S[2][C]
.             .  .
.              . .
.               ..
S[R][1]S[R][2]...S[R][C]

Output Format

A single line containing the minimum number of movements that have to be done for all contestants to be gathered in one rectangle, or -1 if it is impossible to do so.

Explanation of Sample 1

In this sample, all contestants have gathered in one rectangle, so no movement has to be done.

Explanation of Sample 2

In this sample, Pak Dengklek can move the contestant sitting in the first row and the third column to the seat in the third row and the first column. After the move, all contestants have gathered in one rectangle, so only 1 movement has to be done.

Explanation of Sample 3

In this sample, there is no way to gather all contestants in one rectangle.

Subtasks

For all subtasks:

  • 1 ≤ R, C.
  • 1 ≤ R × C ≤ 100,000.
  • S contains only the characters 0 or 1.
  • There is at least one character 1 in S.
  • R = 1.
  • R × C ≤ 2,000.
  • R = 1.
  • R × C ≤ 200.
  • R × C ≤ 2,000.
  • No additional constraints.
